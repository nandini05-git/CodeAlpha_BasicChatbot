# Hangman Game

## Project Overview

This project is a simple text-based Hangman Game developed using Python as part of the CodeAlpha Python Programming Internship.

The game randomly selects a word from a predefined list, and the player must guess the word one letter at a time. The player has a limited number of incorrect attempts before the game ends.

## Features

* Random word selection
* User-friendly text interface
* Letter-by-letter guessing
* Limited incorrect attempts
* Win and lose conditions
* Input validation

## Technologies Used

* Python

## Concepts Applied

* Loops
* Conditional Statements
* Lists
* Strings
* Random Module
* User Input Handling

## How to Run

1. Open the project folder.
2. Open a terminal in the folder.
3. Run the following command:

python hangman.py

## Sample Output

The game displays hidden letters using underscores and updates the word as the player correctly guesses letters.
